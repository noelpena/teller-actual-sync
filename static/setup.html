Need Help?

Finding Your Sync ID

  1. Open Actual Budget
  2. Go to Settings (gear icon)
  3. Click "Show Advanced Settings"
  4. Copy the Sync ID

Finding Your Account ID

  1. Open the account you want to sync in Actual Budget
  2. Look at the browser URL bar
  3. Copy the ID after /accounts/
  4. Example: /accounts/abc123-def456

Server URL Examples

  • http://localhost:5006 - Local installation
  • http://192.168.1.100:5006 - Local network
  • https://actual.yourdomain.com - Remote server
